# Roadmap - RepoLens AI

RepoLens is focused on one product promise: make AI coding agents work with
better project context, clearer prompts, and safer post-edit checks.

## Completed

- v1.0: Repository scanning, framework detection, architecture analysis, risk
  scanning, AGENTS.md generation, and markdown reports.
- v2.0: Interactive context setup, context quality scoring, cross-tool sync,
  framework templates, Cursor skill generation, dashboard, and doctor command.
- v3.0: AST-powered analysis, drift detection, circular dependency detection,
  complexity signals, naming convention detection, and unused export tracking.
- v3.1: Daily vibe-coding loop with `context`, intent-aware `prompt`, `check`,
  and `vibe` readiness scoring.

## Near-Term

- Improve `prompt` with more intent profiles and framework-specific file
  suggestions.
- Add JSON output for `context`, `check`, and `vibe` so teams can automate
  RepoLens in CI.
- Expand real-world examples for Cursor, Antigravity, Claude Code, Codex, and
  GitHub Copilot.
- Add more framework templates for Astro, Remix, SvelteKit, Spring Boot, and
  Rails.
- Add contribution docs for community-submitted templates and skill files.

## Later

- GitHub Action for context drift and AI-readiness checks on pull requests.
- Config file support through `.repolensrc`.
- Multi-repo analysis for teams.
- Custom scoring rules for organizations.
- Optional web dashboard for historical context quality.

## Community

Open an issue with your AI coding workflow, IDE, framework, and the context
