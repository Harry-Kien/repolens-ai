# Roadmap — RepoLens AI

## ✅ v1.0 — Foundation (Completed)

- [x] 7 core CLI commands (analyze, arch, explain, review, risks, onboard, agents)
- [x] Static analysis engine (works without AI)
- [x] Framework detection (15+ frameworks)
- [x] Architecture inference
- [x] Risk scanning (security, architecture, testing)
- [x] Git diff review
- [x] AGENTS.md generation
- [x] Privacy-first design
- [x] Optional AI enhancement via OpenAI
- [x] Markdown report export

## ✅ v2.0 — AI Context Intelligence Platform (Current)

- [x] Interactive AGENTS.md creation (`init` with interview engine)
- [x] 5-dimension context quality scoring (`lint`)
- [x] Auto-fix generic rules and boilerplate (`fix`)
- [x] Cross-tool sync: AGENTS.md → CLAUDE.md → .cursorrules → Copilot (`sync`)
- [x] 12 framework templates with real tribal knowledge (`templates`)
- [x] 7 skill files for common vibe coding tasks (`skills`)
- [x] AI development health check (`doctor`)
- [x] Local web dashboard with zero dependencies (`dashboard`)
- [x] Content-aware code analysis (`contentReader`)
- [x] 15 total CLI commands
- [x] Vietnamese + English documentation

## 🔜 v2.1 — Community & Polish

- [ ] `repolens watch` — Watch mode: auto-lint on AGENTS.md save
- [ ] `repolens diff` — Show what changed since last sync
- [ ] More templates: Svelte, Astro, Remix, Go, Rust, Spring Boot, Rails
- [ ] More skills: Testing, Git workflow, Code review, Performance
- [ ] Template contribution system (community can submit templates)
- [ ] `--lang vi` / `--lang en` flag for bilingual CLI output
- [ ] `repolens lint --fail-on-warning` for CI/CD integration
- [ ] JSON output format (`--format json`)

## 🔜 v2.2 — IDE & CI Integration

- [ ] VSCode extension (visual AGENTS.md editor)
- [ ] GitHub Action for PR context review
- [ ] Pre-commit hook support (`repolens lint` on commit)
- [ ] Cursor extension (auto-sync on save)
- [ ] `.repolensrc` configuration file

## 🔜 v3.0 — Team & Enterprise

- [ ] Team onboarding mode (multi-repo context management)
- [ ] Shareable HTML reports
- [ ] Historical analysis (track context quality over time)
- [ ] Multi-AI model support (Anthropic, Google, local LLMs)
- [ ] Plugin system for custom analyzers
- [ ] API for programmatic access
- [ ] SaaS dashboard (optional cloud tier)
